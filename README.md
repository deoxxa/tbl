tbl - tiny bencode library
==========================
*tbl* is a tiny event driven [bencode](http://en.wikipedia.org/wiki/Bencode)
library written in C. Its api is similar to to the api of
[yajl](http://github.com/lloyd/yajl).

It's supposed to be

* fast
* small
* easy to use

Compilation
-----------

	./waf configure
	./waf

### Run Test

	./build/default/test
